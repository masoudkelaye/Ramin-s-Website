# سایت رامین قربانی — راهنمای راه‌اندازی

## تغییرات جدید در این نسخه

- **عکس‌ها ادیت حرفه‌ای شدن:** همه‌ی عکس‌های گالری یکدست و مشکی‌سفید با کنتراست و شارپ‌نس استودیویی شدن؛ عکس اصلی (hero) هم رنگی ولی با گریدینگ ملایم‌تر و حرفه‌ای‌تر.
- **گالری داینامیک شد:** به‌جای زیر هم اومدن، عکس‌ها به‌صورت افقی (اسکرول کن ببین) و با اندازه‌های نامنظم نمایش داده میشن — یه راهنمای کوچیک «برای دیدن بیشتر بکش» هم بالاشه.
- **انیمیشن قیچی:** سمت راست صفحه یه خط باریک هست که موقع اسکرول کردن، یه قیچی روش حرکت می‌کنه و واقعاً «می‌بره» (بلیدهاش باز و بسته میشن) — نشون‌دهنده‌ی میزان پیشرفت اسکرول توی صفحه.
- **قابلیت اپلیکیشن (PWA):** وقتی کاربر از مرورگر موبایل گزینه‌ی «Add to Home Screen» رو بزنه، سایت با آیکون اختصاصی (مونوگرام RG) به‌صورت فول‌اسکرین (بدون نوار آدرس) باز میشه.
  - توی اندروید/کروم این کاملاً فول‌اسکرینه.
  - توی آیفون/سافاری، اپل اجازه‌ی فول‌اسکرین ۱۰۰٪ نمیده ولی نوار وضعیت (ساعت/باتری) شفاف و مینیمال میشه که خیلی نزدیک به فول‌اسکرینه — این محدودیت خود اپله، نه سایت.

## ۱. آپلود روی Vercel (رایگان)

⚠️ **نکته امنیتی مهم:** لطفاً هیچ‌وقت رمز عبور یا اطلاعات ورود حساب‌های آنلاین (Vercel, Gmail, Instagram و ...) رو برای هیچ‌کس، حتی من، نفرستید. من نمی‌تونم و نباید وارد حساب شما بشم.

از همون روش قبلی استفاده کن (آپلود فایل‌ها به گیت‌هاب با لینک‌های `upload/main/...` که قبلاً گفتم)؛ این‌بار به‌جز فایل‌های قبلی، این‌ها هم باید آپلود بشن:
- `manifest.json` → مستقیم توی ریشه‌ی پروژه (کنار index.html)
- پوشه‌ی `assets/icons/` → شامل ۴ فایل آیکون (از لینک `.../upload/main/assets/icons`)
- عکس‌های ادیت‌شده‌ی جدید توی `assets/img/` باید جایگزین نسخه‌های قبلی بشن (همون اسم فایل‌ها، فقط محتوای جدید)

## ۲. راه‌اندازی رزرو نوبت (Cal.com)

1. برو به [cal.com](https://cal.com) و رایگان ثبت‌نام کن.
2. توی تنظیمات (Settings → Apps) گوگل کلندرت رو وصل کن.
3. دو تا Event Type بساز:
   - **Normal** با مدت زمان ۳۰ دقیقه
   - **VIP** با مدت زمان ۶۰ دقیقه
4. ساعت کاری رو روی هر روز ۹:۰۰ تا ۱۹:۰۰ تنظیم کن.
5. توی تنظیمات نوتیفیکیشن، ایمیل و اپلیکیشن موبایل Cal.com رو فعال کن تا نوتیفیکیشن روی گوشیت بیاد.
6. توی فایل `index.html` دنبال بخش `<!-- CAL.COM SETUP -->` بگرد و کد embed رو با یوزرنیم خودت جایگزین کن (توضیحش دقیقاً همونجا نوشته شده).

اگه بخوای، همین الان که یوزرنیم Cal.com رو ساختی بهم بگو تا خودم کد embed رو برات توی فایل جایگزین کنم.

## ۳. تغییر عکس‌ها یا متن‌ها بعداً

- عکس‌ها توی پوشه `assets/img/` هستن؛ فقط با عکس جدید (با همون اسم فایل) جایگزینشون کن.
- متن‌های هر زبان توی فایل `i18n.js` هستن، هر زبان یه بخش جدا داره.

## فایل‌های پروژه
```
ramin-site/
├── index.html      → ساختار اصلی سایت
├── style.css        → طراحی و ظاهر
├── script.js         → سوییچ زبان
├── i18n.js           → متن‌های ۱۰ زبان (آلمانی، انگلیسی، فارسی، ترکی، اوکراینی، روسی، فرانسه، اسپانیایی، ایتالیایی، لهستانی)
└── assets/img/       → عکس‌های نمونه‌کار
```
