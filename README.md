# سایت رامین قربانی — راهنمای راه‌اندازی

## ⚠️ کار فوری: توکن گیت‌هابت رو باطل کن

توکنی که برام فرستادی رو همین الان استفاده کردم و همه‌چی رو آپلود کردم، ولی چون توی چت به‌صورت متن ساده فرستاده شده، دیگه بهش به چشم یه رمز لو رفته نگاه کن. لطفاً:
1. برو به [github.com/settings/tokens](https://github.com/settings/tokens)
2. توکنی که ساخته بودی رو پیدا کن و **Delete** بزن (باطلش کن)
3. یه توکن جدید بساز (پایین توضیح دادم دقیقاً چطوری) — این توکن جدید رو دیگه هیچ‌جا (نه اینجا، نه جای دیگه) نفرست؛ فقط داخل تنظیمات Vercel وارد میشه.

## بخش جدید: پنل مدیریت (Admin Panel)

حالا یه پنل مدیریت داری که میتونی بدون نیاز به من:
- ✅ ۴ تا آمار (مثل «۱۲+ سال تجربه») رو وارد/ویرایش کنی
- ✅ نظرات مشتری‌ها رو اضافه، ویرایش یا حذف کنی
- ✅ هر کدوم از عکس‌های گالری یا عکس اصلی (hero) رو با عکس جدید جایگزین کنی

آدرس پنل بعد از راه‌اندازی: `https://آدرس-سایتت/admin/`

**محدودیت فعلی:** ویرایش متن‌های عمومی سایت (مثل توضیحات زیر اسم، یا خدمات) هنوز از طریق پنل امکان‌پذیر نیست — اگه بعداً خواستی این قابلیت رو هم اضافه کنم بگو.

### راه‌اندازی پنل مدیریت (فقط یک‌بار لازمه)

برای اینکه پنل بتونه تغییرات رو ذخیره کنه، باید ۴ تا متغیر محیطی (Environment Variable) توی تنظیمات Vercel وارد کنی:

**۱. ساخت توکن جدید گیت‌هاب:**
- برو به [github.com/settings/tokens?type=beta](https://github.com/settings/tokens?type=beta) (Fine-grained tokens)
- **Generate new token** بزن
- یه اسم بذار (مثلاً `ramin-site-admin`)
- توی «Repository access» فقط ریپازیتوری `Ramin-s-Website` رو انتخاب کن (نه همه‌ی ریپوها)
- توی «Permissions» → «Contents» رو بذار روی **Read and write**
- Generate token بزن و کدشو کپی کن (فقط یه‌بار نشونش میده)

**۲. وارد کردن متغیرها توی Vercel:**
- برو به داشبورد Vercel → پروژه‌ی سایتت → **Settings → Environment Variables**
- این ۴ مورد رو یکی‌یکی اضافه کن:

| نام | مقدار |
|---|---|
| `GITHUB_TOKEN` | همون توکنی که بالا ساختی |
| `GITHUB_REPO` | `masoudkelaye/Ramin-s-Website` |
| `GITHUB_BRANCH` | `main` |
| `ADMIN_PASSWORD` | یه رمز قوی و دلخواه برای ورود به پنل |

- بعد از اضافه‌کردن، از تب **Deployments**، روی آخرین دیپلوی، منوی سه‌نقطه → **Redeploy** بزن تا متغیرها اعمال بشن.

بعدش برو به `آدرس-سایتت/admin/`، رمزی که خودت انتخاب کردی رو وارد کن، و شروع کن به مدیریت.

⚠️ رمز پنل ادمین رو جایی امن نگه‌دار (نه توی چت با من). اگه فراموشش کردی، کافیه توی Vercel مقدار `ADMIN_PASSWORD` رو عوض کنی.

## تغییرات این نسخه

- استایل کاملاً بازطراحی شد: تم مشکی و طلایی لوکس با فونت سریف ایتالیک، الهام‌گرفته از یه نمونه‌ی مرجع
- بخش آمار و نظرات مشتری‌ها اضافه شد (فعلاً خالی — از پنل مدیریت پرش کن)
- پنل مدیریت با اتصال مستقیم به گیت‌هاب (بدون نیاز به دیتابیس جداگانه)

## ۱. آپلود روی Vercel (رایگان)

⚠️ **نکته امنیتی مهم:** لطفاً هیچ‌وقت رمز عبور یا اطلاعات ورود حساب‌های آنلاین (Vercel, Gmail, Instagram و ...) رو برای هیچ‌کس، حتی من، نفرستید. من نمی‌تونم و نباید وارد حساب شما بشم.

از همون روش قبلی استفاده کن (آپلود فایل‌ها به گیت‌هاب با لینک‌های `upload/main/...` که قبلاً گفتم)؛ این‌بار به‌جز فایل‌های قبلی، این‌ها هم باید آپلود بشن:
- `manifest.json` → مستقیم توی ریشه‌ی پروژه (کنار index.html)
- پوشه‌ی `assets/icons/` → شامل ۴ فایل آیکون (از لینک `.../upload/main/assets/icons`)
- عکس‌های ادیت‌شده‌ی جدید توی `assets/img/` باید جایگزین نسخه‌های قبلی بشن (همون اسم فایل‌ها، فقط محتوای جدید)

## ۲. رزرو نوبت

سیستم رزرو آنلاین حذف شد؛ به‌جاش دکمه‌ی اصلی سایت و بخش تماس مستقیم به شماره‌ی تلفن/واتساپ رامین وصله (`tel:` و `wa.me`) تا مشتری‌ها مستقیم تماس بگیرن یا پیام بدن.

## ۳. تغییر عکس‌ها یا متن‌ها بعداً

- عکس‌ها توی پوشه `assets/img/` هستن؛ فقط با عکس جدید (با همون اسم فایل) جایگزینشون کن.
- متن‌های هر زبان توی فایل `i18n.js` هستن، هر زبان یه بخش جدا داره.

## فایل‌های پروژه
```
ramin-site/
├── index.html      → ساختار اصلی سایت
├── style.css        → طراحی و ظاهر
├── script.js         → سوییچ زبان
├── i18n.js           → متن‌های ۱۰ زبان (آلمانی، انگلیسی، فارسی، ترکی، اوکراینی، روسی، فرانسه، اسپانیایی، ایتالیایی، لهستانی)
└── assets/img/       → عکس‌های نمونه‌کار
```
